# AQI Canada Prediction

Predicting Air Quality Index (AQI) values across Canadian cities using machine learning.

## Project Structure

```
aqi-canada-prediction/
├── data/           # Raw and processed datasets
├── notebooks/      # Exploratory and modelling notebooks
├── src/            # Source modules
├── models/         # Saved trained models
├── results/        # Metrics and plots
├── demo/           # Quick prediction demo
└── report/         # Final report
```

## Setup

```bash
pip install -r requirements.txt
```

## Usage

Run notebooks in order (01 → 04) for the full pipeline, or use `demo/predict_sample.py` for a quick prediction.
